9.관계자외 분해결합 금지.
10.교육 받지 않은 인원 사용금지.

{테이블 쏘 안전수칙 12가지}

1. 장갑을 찐채로 테이블쏘를 사용하지 말것.
2. 작업 변경 내 바닥을 걸리적거리는 것 없이 깨끗이 치울 것.
3. 보안경과 귀마개 착용
4. 소매가 길게 늘어지는 옷이나 넥타이 긴 장신구등을 하지 말것
5. 안정적으로 서서 작업하고 미끄럽지 않은 신발을 신을 것
6. 확신이 없는 작업은 피할 것
7. 15cm이하의 너비를 가진 부재를 자를 때는 푸쉬스틱을 사용할 것
8. 짧은 부재를 자를 때는 스탑블럭을 사용할 것
9. 톱날과 일직선상에 서 있지 말것
10. 톱날이 돌아가고 있을 때는 절대 톱날 뒤쪽으로 손을 가져가지 말것
11. 톱날이 교체하거나 수리할 때는 전원 플러그를 빼고 할 것
12. 테이블쏘를 세팅하기 전에 반드시 톱날이 멈췄는지 확인할 것
13. 테이블쏘를 켜기 전 톱날이 잘 움직이는 지 확인할 것
14. 정반을 작업물이 잘 미끄러지도록 부드럽게 관리할 것
15. 립펜스와 톱날이 평행하도록 할것(톱날에 작업물이 끼지 않아 킥백을 방지함)
16. 제로 클리어런스 인써트를 사용할 것
17. 블레이드 인써트 없이 절대 테이블쏘를 켜지 말것
18. 프리핸드컷을 하지 말것
19. 블레이드 가드, 라이빌나이프, 폴을 반드시 장착할 것
20. 작업물이 톱날을 완전히 지나갈 때까지 잡고 있을 것
21. 자르던 부재가 들리거나 정반에서 뒤틀려서 손에서 놓치면 테이블쏘 전원을 끌것
22. 자르기 전에 나무를 살펴볼 것(못이나 돌이 박신 곳이나 옹이는 없는지)
23. 립펜스와 마이터게이지를 함께 사용하지 말것
24. 테이블쏘가 켜져 있을 때는 펜스를 움직이지 말것
